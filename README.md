# Loan-Default-Prediction

Loan default prediction is an important problem in many banks or financial institutions as losses may be incurred if the borrowers default. 
This repository consists of the codes for a data science challenge for loan default prediction on a real world data set provided by Coursera. In this project, different classifiers(logistic regression, random forests, XGBoost and LightGBM) are applied to the data set and the results are compared among different classifiers.
